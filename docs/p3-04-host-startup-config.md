# P3-04: ホスト起動設定の集約（フォント・ウィンドウタイトル、§9.1 P3）

設計書 §9.1 P3-04 に基づき、環境依存の起動設定を `main` 直下のホストモジュールへ集約した。

## 変更内容

| 項目 | 変更 |
|------|------|
| 新規モジュール | `src/app_host.rs` |
| 集約対象 | `APP_WINDOW_TITLE`、初期ウィンドウサイズ、`configure_japanese_font` の初期化と警告反映 |
| `main.rs` | `build_native_options()` と `apply_host_startup_effects(...)` を呼ぶだけの薄い起動コードへ変更 |

## 既存挙動との互換

- タイトル文字列（`条例分析ビューア`）と初期サイズ（`1480x920`）は変更しない。
- フォント未検出時の警告文言・表示条件（`error_message` が空のときのみ設定）も維持する。

## 手動確認（完了条件）

1. 起動時のウィンドウタイトル・初期サイズが従来どおり。
2. 日本語フォントが無い環境で警告が従来どおり出る。
